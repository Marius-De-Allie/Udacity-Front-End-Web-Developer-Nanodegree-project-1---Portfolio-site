Dice Game

Created using: JavaScript | HTML | CSS

JavaScript

Dice Game is a web based game, where the objective is to try to get to the winning score of 100 points first by rolling the dice. If the current player gets a dice role of 1, then they lose a turn and control goes over to the other player. The current player can chose to hold their currently accumaletated score for that round and by doing so gives control to the other player. This app harnesses the power of JavaScript event handlers and DOM manipulation to achieve it's core functionality.
